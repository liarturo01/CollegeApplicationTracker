# College Application Tracker

A simple application to organize, track, and manage college applications in one place.

## Overview

College Application Tracker helps students keep track of multiple college applications by storing key information such as application status, deadlines, notes, and decisions. The goal is to reduce missed deadlines and make the application process easier to manage.

## Features

* Track multiple colleges in one centralized list
* Store application statuses (e.g., Not Started, In Progress, Submitted, Accepted, Rejected)
* Keep notes for each application
* Simple and readable project structure
